# Rick and Morty API Test

This is a test I made of the Rick and Morty GraphQL API. 
I queried the name and image of each character in Rick and Morty and displayed them on pages of 20 characters. 

## Features
- Fetches character name 
- Fetches and displays character image
- Pages

## API
https://rickandmortyapi.com/documentation
https://rickandmortyapi.com/graphql
